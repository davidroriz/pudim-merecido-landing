# Arquitetura Simples para Landing Page de Produto

## Estrutura de Arquivos

```
landing-page/
├── index.html
├── style.css
├── script.js
└── images/
    ├── produto.jpg
    ├── logo.png
    └── background.jpg
```

## Tecnologias

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e responsividade
- **JavaScript Vanilla** - Interatividade básica
- **Hospedagem** - GitHub Pages, Netlify ou Vercel (gratuito)

## Seções da Landing Page

### 1. Header (Cabeçalho)
- Logo da marca
- Menu de navegação simples (Produto, Benefícios, Comprar)

### 2. Hero Section (Seção Principal)
- Imagem grande e atraente do produto
- Título impactante
- Subtítulo descritivo
- Botão CTA (Call-to-Action) "Comprar Agora"

### 3. Sobre o Produto
- Descrição breve e objetiva
- 3-4 características principais
- Ingredientes destacados

### 4. Benefícios
- Lista de 3-5 benefícios principais
- Ícones ou imagens ilustrativas

### 5. Galeria de Imagens
- 3-6 fotos do produto em diferentes ângulos
- Imagens de alta qualidade

### 6. Depoimentos (opcional)
- 2-3 avaliações de clientes
- Fotos e nomes (se disponível)

### 7. Call-to-Action Final
- Preço em destaque
- Botão de compra grande e visível
- Informações de entrega

### 8. Footer (Rodapé)
- Informações de contato
- Redes sociais
- Links legais (Política de Privacidade, Termos)

## Design Principles

- **Mobile-First**: Responsivo para todos os dispositivos
- **Cores**: Relacionadas ao produto (ex: verde para orgânico, vermelho para picante)
- **Tipografia**: Máximo 2 fontes (uma para títulos, outra para texto)
- **Performance**: Imagens otimizadas (WebP, compressão)
- **Acessibilidade**: Contraste adequado, textos alternativos

## Integração de Pagamento (Simples)

- Integração com Mercado Pago/PagSeguro
- Formulário de pedido por email
